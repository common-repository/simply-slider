== Description ==

= Simply Slider Plugin =

Simply Slider provides a simple and easy to use engine for uploading and managing slides of images,
with the ability to add images, add different overlays for each image, change text color for each slide, add text and headline or a logo file.

The Simply Slider doesn't overwhelm you with functions and options, it simply does what you see in the example. Set up a nice responsive slider in a minute.


Demo: https://www.themewerk.com/plugins/

= Simply Slider Plugin Features =


*Upload Images*

* Simply Slider plugin offers easy and fast functionality for getting images from your desktop to your website.
* You can easily upload images via a standard WordPress-style uploader. Add Text/Slogan and/or your Logo image file as an animated element.
* Upload up to 5 Slider images (easy to add more).
* Add your slider with a shortcode in your page/post.

*Manage Simply Slider*

* Slider Options Panel. Enjoy a single location where you can see and manage all your images.
* Add Image
* Add Text.
* Add your logo or Headline.
* Add Overlay
* Choose Text Color for each Image

*Display Slider*

* Insert the slider with a shortcode in a post/page. Shortcode [simplyslider]

== Installation ==

1. 	Download, upload and install .zip under Plugins >> Add New > Upload, and activate the Simply Slider plugin.

2.	From your Wordpress Dashboard, go to Settings > Simply Slider > Add Images & Text > Follow the on-screen cues.

3. 	Go to a post/page, and add the shortcode [simplyslider].

That's it ... have fun! For more information, feel free to visit the official website for the Simply Slider <a href="http://www.simplyslider.com" target="_blank">Simply Slider Plugin</a>.

*Use Shortcode [simplyslider]

== Frequently Asked Questions ==

= Will Simply Slider work with my theme? =

It works with the vast majority of premium and free themes, without requiring any coding.


= Are the galleries flash based? =

No, Simply Slider uses Javascript (jQuery) based displays to ensure compatibility across the widest range of displays possible.


= Is the slider mobile friendly? =

Yes, since we use Javascript rather than flash, Simply Slider is compatible with Android, iOS, and Blackberry.


= Can I add HTML to the captions? =

No, caption areas are for text only, you can style it through css or your theme css to fit the design of your homepage.
